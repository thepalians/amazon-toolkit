# Amazon Seller Toolkit — Chrome Extension

## Features
- 🔍 Quick ASIN lookup from any Amazon page
- 💰 Instant profit calculator (offline)
- 📊 BSR sales estimator (offline)
- 📦 Auto-detects ASIN on Amazon product pages
- 🛒 Floating overlay on Amazon pages with quick stats
- 🔗 Quick links to full dashboard tools

## Installation
1. Open Chrome → `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `chrome-extension` folder
5. Pin the extension to toolbar

## Setup
1. Click the extension icon
2. Go to ⚙️ Settings tab
3. Enter your API URL: `https://palians.com/amazon-seller-toolkit/api`
4. Enter your JWT auth token (from browser localStorage)
5. Save settings

## Usage
- **On Amazon pages**: Click the 🛒 floating button to see product stats
- **Popup**: Use ASIN tab for lookup, Profit tab for calculations, BSR for estimates
